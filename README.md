# Syntecxhub User Authentication API

A Backend Authentication API built using Node.js, Express.js, MongoDB Atlas, bcryptjs, and JWT.

## Features

- User Signup
- User Login
- Password Hashing using bcryptjs
- JWT Authentication
- MongoDB Atlas Database Connection
- REST API

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- bcryptjs
- jsonwebtoken
- dotenv
